# Influencer marketing application
Influencer marketing application backend code in java spring boot

# Current Technologies 

* Spring Boot 3.1.4
* Spring Security
* JWT
* Hibernate
* Maven
* Java 17
* JPA
* H2 Database (For Testing)
* MySQL Datatebase
* REST API
